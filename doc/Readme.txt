===============================================================================

　Kirara (動画プレイヤ試作)

===============================================================================


■ソフトの概要

　動画プレイヤの試作版です。
　色々と難がありますが、いちおう動画プレイヤとして機能します。

　ogv, ogg 形式しか再生出来ません。
　mp4, mp3 などそれ以外の動画・音楽形式にも対応していますが、
　再生前に(内部的に) ogX 形式に変換する処理が入るため、再生開始まで時間が掛かります。


　★当ソフトは ffmpeg を使用します。(外部コマンドとして呼び出します)
　　当ソフト使用前に ffmpeg を導入していただく必要があります。


■動作環境

　Windows 10 Pro または Windows 10 Home


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■ffmpeg導入方法

　1. ブラウザから https://ffmpeg.org/ を開いて下さい。

　2. ページ左側のメニューの Download をクリックして下さい。

　3. Get the packages の Windows のロゴにカーソルを合わせ、
　   Windows Packages の Windows Builds をクリックして下さい。

　4. Version の 3.2.4 を選択して下さい。

　　★バージョン番号が更新されている場合は Previous Builds からダウンロードして下さい。
　　　最新版で代替することも可能ですが、当ソフトが正常に動作しないかもしれません。

　5. Architecture の Windows 64-bit または Windows 32-bit を「導入環境に合わせて」選択して下さい。
　   よくわからない場合は Windows 32-bit を選択して下さい。

　6. Linking の Shared を選択して下さい。(Static でも OK です)

　7. Download Build をクリックして .zip ファイルをダウンロードして下さい。

　8. ダウンロードした .zip を任意の場所に展開して下さい。
　   展開先のパスは後で使用するので、覚えておいて下さい。


　★この手順は 2017/3/13 に作成しました。(2019/7/8 修正)

　★開発・テスト環境では ffmpeg-3.2.4-win64-shared.zip を使用しています。

　★開発・テスト環境では C:\app\ffmpeg-3.2.4-win64-shared に展開しました。


■起動方法

　Kirara.exe を実行して下さい。


■基本的な使い方

　●普通に mp4 mp3 などを再生する

　　1. プレイリストに動画・音楽ファイルをドラッグ・アンド・ドロップする。

　　2. 変換が終わるまで待つ。

　　3. 変換が終わったら再生する。


　●事前に ogX 形式に変換してから再生する

　　1. コンバート / コンバートするファイルを追加(A) を開く。

　　2. Conversion Dialog に動画・音楽ファイルをドラッグ・アンド・ドロップする。

　　3. 変換が終わるまで待つ。

　　4. 変換された ogX 形式のファイルをプレイリストにドラッグ・アンド・ドロップして再生する。


■その他の使い方

　●再生する

　　プレイリスト上の項目をダブルクリックするか、
　　プレイリスト上の項目を右クリックして「再生」を選択するか、
　　プレイリスト上の項目を選択してスペースキーを押すか、
　　スクリーン左下の再生ボタンを押して下さい。

　●再生位置を変更する

　　スクリーン下のシークバーを操作して下さい。
　　(音楽はシーク出来ません)

　●音量を調整する

　　スクリーン右下のゲージで調整出来ます。
　　(動画と音楽で加減が異なるという問題があります)

　●スクリーンを全画面表示にする

　　スクリーン右上の黄色いボタンを押すか、スクリーン中央をダブルクリックして下さい。

　●スクリーンのサイズを変更する

　　スクリーンの端をドラッグして調整するか、
　　プレイリスト上部メニューの、スクリーン(S) / 位置・サイズ変更(S) 又は、サイズ(T) から変更するか、
	プレイリスト上部メニューの、スクリーン(S) / サイズ(T) から適当なサイズを選んで下さい。

　●プレイリストをファイルに保存する

　　プレイリスト上部メニューの、プレイリスト(L) / ファイルに保存(A) から保存して下さい。

　●プレイリストをファイルから読み込む

　　プレイリスト上部メニューの、プレイリスト(L) / ファイルから読み込む(A) から読み込んで下さい。

　●終了する

　　プレイリストを閉じるか、
　　スクリーン右上の赤いボタンを押して下さい。


■制限

　700 MB を超える動画・音楽ファイルは読み込めません。


■補足

　ログファイル Kirara.log, Kirara0.log は、合計およそ 2 MB を超えないように管理（自動的に削除）されます。

　スクリーン左上にちょろちょろ出る文字は、
　設定(S) / スクリーン / インスタント・メッセージを表示しない。のチェックを外すと消えます。
